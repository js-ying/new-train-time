# 新-台鐵時刻表

查詢台鐵火車、高鐵、桃園機場捷運時刻表，簡單又快速。即時列車通阻、即時誤點、歷史快查、票價查詢、訂票導頁功能，讓通勤或旅行更優雅。

此為新版，以 react 18 + next.js + tailwindCSS 3 打造。

# NEW - Taiwan Railway Timetable

Easily check Taiwan Railways, High-Speed Rail, and Taoyuan Airport MRT schedules. Get real-time train status, delay alerts, historical search, fare inquiries, and ticket booking guidance for a smoother commute or trip.

It is new version. Built in react 18 + next.js + tailwindCSS 3.
